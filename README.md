# Karuta Datemap Automation
Python built automation to solve datemap mini game in karuta bot discord.

# How to use
- Install necessary programs (python, pyautogui library etc).
- Run the program, paste the solution emoji from any karuta datemap solver bot. DONE :white_check_mark: 
- The pasted emoji format must have a single space in between such as `:arrow_up_small: :blossom: :arrow_up_small: ...`
- The discord channel must only have 1 datesolver bot in order to not break the program detection area.
- Do whatever you want with this code. Have fun.
